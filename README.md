# TwitterClone

Twitter clone built with Django Rest Framework, PostgreSQL, Djoser/SimpleJWT for JWT Auth, NextJS, Redux Toolkit-Query, Redux Persist, React-Bootstrap, and Scss modules.

Features include:
 - user registration
 - user auth
 - user verification
 - account recovery
 - editable profiles
 - posts
 - feeds
 - direct messaging
 - posts
 - groups
 - events
